General Settings

Default Behavior
This sets the behavior for sites without a per-site setting (see below).

Hero Mode -- Do nothing, leave sites alone
Zero Mode -- Remove the hero image (recommended)

Indication
This is what happens when a hero image is zerod. Note that when using the no indication mode you will lose the ability to click the placeholder to restore the hero.

No indication, just remove
Show a placeholder (recommended)

Detection Level
There are various factors that go into detection, but one of them falls back on absolute image size. This sets the minimum size of an image that can be consider a hero. It must meet both requirements.

At least pixels wide
At least pixels tall

Note: (this is preliminary, a future version will change to a Low/Medium/High sensitivity factor)

Per-Site Settings
These settings will override the default behavior for a specific site.

When you visit a site and click on the toolbar icon to toggle Hero/Zero mode that setting will be noted here. You can remove a site from the list and it will go back to following the default behavior (see above).
